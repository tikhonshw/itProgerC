    *.sass

    config/
3. создаем локальный репозиторий: git init - инициализация проекта
4. добавляем все файлы проекта в локальный репозиторий: git add .
5. добавляем фиксацию и описание загрузки: git commit -m "Описание"
6. проверка git Status
7. создаем новую ветку: newBranch
